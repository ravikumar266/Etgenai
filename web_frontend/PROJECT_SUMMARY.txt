================================================================================
                            🎨 THEME COLORS
================================================================================

LIGHT MODE:
   Background:     Off-white      (oklch(0.98 0 0))
   Foreground:     Dark gray      (oklch(0.15 0 0))
   Card:           Pure white     (oklch(1 0 0))
   Accent:         Amber/Gold     (oklch(0.72 0.2 70))
   Border:         Light gray     (oklch(0.88 0 0))
   Success:        Muted emerald  (oklch(0.56 0.15 150))
   Destructive:    Muted rose     (oklch(0.53 0.17 30))

DARK MODE:
   Background:     Deep obsidian  (oklch(0.1 0 0))
   Foreground:     Off-white      (oklch(0.98 0 0))
   Card:           Dark neutral   (oklch(0.13 0 0))
   Accent:         Vibrant amber  (oklch(0.72 0.2 70)) [Same as light]
   Border:         Subtle dark    (oklch(0.18 0 0))
   Success:        Muted emerald  (oklch(0.5 0.12 150))
   Destructive:    Muted rose     (oklch(0.5 0.15 30))

KEY DESIGN DECISIONS:
   ✓ Amber accent consistent across both modes
   ✓ OKLch color space for perceptual uniformity
   ✓ Separate sidebar tokens for visual hierarchy
   ✓ High contrast for accessibility
   ✓ Status colors optimized for readability

================================================================================
                          🚀 QUICK START GUIDE
================================================================================

1. INSTALL & RUN:
   npm install
   npm run dev
   → Open http://localhost:3000

2. SEE THEMES:
   Click theme icon (bottom-left sidebar)
   • ☀️ Light
   • 🌙 Dark
   • 🖥️ System

3. EXPLORE FEATURES:
   • Chat with AI
   • Approve/deny emails
   • Upload documents
   • Manage knowledge base
   • View chat history

4. CUSTOMIZE:
   Edit app/globals.css to change colors
   Edit components/axiom/* for structure
